# team-slider

## Tech stack used:

- JS (ES6)
- GULP
- SASS
- SWIPER

## Getting started

Edit team members info in `/data/team.js` file.

```js
{
    firstName: 'Basia',
    lastName: 'Sołtysińska',
    role: 'Founder',
    location: 'Warsaw',
    imageURL: './assets/person1.png',
    email: '',
}
```

You can specify the email, if not the default email will be `firstName.lastName@indahash.com`
